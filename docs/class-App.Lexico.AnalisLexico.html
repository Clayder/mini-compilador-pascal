Percorre o código criando os tokens, a partir do caracter atual (chAtual). Quando terminar de formar o token em relação ao caracter atual, o próximo token é escolhido (atualiza chAtual e idAtual, essa atualização é feita no "do while" do método analisarLetras da classe Letra) com issp método (nextToken) é finalizado. Depois de finalizar o método o programa volta para a main (compilador.php), dando continuidade na execução do “do while”.
